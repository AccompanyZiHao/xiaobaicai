hexo 创建文章、标签、分类

  • hexo
  • 2018-11-10

1.创建文章

    在hexo下创建一个新的文章

hexo new ‘文章名称’

创建成功后悔提示你创建在文件路径，一般都在 hexo/source/_posts 下

    文章的基本设置

title: 文章名字
date: 2018年11月10日14:14:13
comments: true #是否可以评论
toc: true # 是否显示文章目录
categories: ‘hexo创建文章的基本设置’ #分类
tage: #标签

2.创建标签、分类

    创建标签页面

hexo new page tags

    创建分类

hexo new page categories
